5 Design Principles of Netflix

Affordance

The intended function of the circle/triangle in the middle is served as a "play" button. Also, the "X" in the top right corner serves as the close out button to take you back to the main menu. Each of these buttons are clear to the user that you have two options, you can either play the movie or exit out to continue browsing.

Chunking

This section is a sub-section that allows the user to discover the genres that are available on Netflix. This is convenient so the user can narrow down their searching.

Hicks Law

This section of the website is clear to the user that you can scrub through to discover content. The many options for movies serve as the menu.

Mapping

The location of these options are at the top of the screen to remind users they can always scroll back up to the top to hit the main menu, or execute main functions such as; logging out, switching users, and selecting a genre.

Modularity

This section of the website is a clear example of the different modules which are the movies themselves. Giving the user a brief overview of what the movie is about.
